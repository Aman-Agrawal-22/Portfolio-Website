My Work

Waves Of Food

"Waves Of Food" is a dynamic food ordering app developed using Kotlin and XML. It includes features such as user sign-up, cuisine exploration, cart management, order tracking, and profile updates. The app offers an intuitive interface for users to browse and place orders with ease, enhancing the overall food ordering experience.

Parkinson's Prediction Model

The Parkinson’s Prediction Model predicts the progression of Parkinson's disease using machine learning algorithms. Built with Python and various machine learning frameworks, it leverages datasets to provide accurate predictions on disease progression and UPDRS scores, helping in better healthcare decision-making.

N-Queens Visualizer

The N-Queens Visualizer is a project designed to showcase the backtracking algorithm used to solve the N-Queens problem. The interactive visualizer allows users to input values for N and watch the algorithm in action as it places queens on the chessboard while avoiding conflicts.
